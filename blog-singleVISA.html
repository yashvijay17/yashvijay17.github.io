MY TIME AT VISA INC.

I joined VISA as a Software Engineer, coming staright out of VIT as an undergraduate student. We were called the NCGs - New College Graduates.

I started as a Developing REST APIs and then was shifted to the Data Engineering team where I was introduced to ETL and the tool Talend.

My Tech Stack was predominantly - Talend, JAVA, MySQL, MongoDB along with essential technical tools and languages.

As a part of the Data Services team, I was responsible for developing Talend ETL jobs for the Global Opcerts project which involved extracting data from different streams and file sources like file systems, DBs, and APIs etc. and then transforming them by applying business rules (Using components) and loading the data onto various platforms.
Adept at TAC (automation of job loads and management), Bulk loads, data manipulations etc.
I lead proof of concept (POC) projects, working on Web Apps (Tech stack being React JS, Rest APIs and MySQL), API calibration from different sources and integrating those data into MongoDB and croning the jobs (Triggers).
I Mentored interns for their summer internship projects - Lead of Data Integration and Data Services which resulted in them converting their internship into fulltime.
As an active participant in VISA CSR events, I took part in various events which gave back to the community, helped the underpriveledged and impart education to the youth.
Lead the VISA Football Club for the Bangalore centre.
